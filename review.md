            - ' _ ' (언더바)는 와일드카드라고 부르고 앞에 조건들을 제외한 모든 것을 포함
            - ' | ' (OR)은 또는을 의미. 둘 중에 하나라도 만족하면 참
            - ' * ' (별)은 앞에 값을 제외하고 나머지 요소들을 모두 저장하는 변수(패킹) 앞에 붙임
- match~case문
```
match 변수 or 값:
    case 값:
        실행문
```

- **Result** – 결과
아래 내용을 이해함

        1. flag 변수 - 여러개의 조건문들이 하나의 상황에 의해서 함께 제어되어야 하는 경우
        2. := (왈러스 오퍼레이터, 바다코끼리 연산자) - 할당과 리턴을 동시에 하게 해줍니다.
